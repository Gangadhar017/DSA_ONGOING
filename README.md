# DSA Ongoing

This repository contains Data Structures and Algorithms (DSA) practice problems and solutions, organized by lecture topics.

## Structure

The repository is organized into lecture folders (Lec1, Lec2, etc.), each containing relevant Java files and PDFs for the topics covered.

## Topics Covered

- Basic Programming Concepts
- Arrays and Strings
- Recursion
- Data Structures (Stacks, Queues, Linked Lists, Trees, Graphs, etc.)
- Algorithms (Sorting, Searching, Dynamic Programming, etc.)
- Bit Manipulation
- And more...

## Usage

Each lecture folder contains Java code examples and problem solutions. PDFs may include lecture notes or problem statements.

Feel free to explore and use this repository for learning and reference.
